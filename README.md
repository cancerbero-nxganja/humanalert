## What is HumanAlert?

HumanAlert is a self-improving platform with three modules:

| Module | Function |
|--------|----------|
| **AlertRed** | Real-time emergency alerts for natural disasters |
| **BuscaVida** | Global missing persons network (open Amber Alert) |
| **MapaSolidario** | Collaborative relief coordination map |

## Quick Start (5 minutes)

```bash
git clone https://github.com/cancerbero-nxganja/humanalert.git
cd humanalert
cp .env.example .env
docker-compose up -d
```

Open http://localhost:3000

## Modules

### AlertRed
Real-time emergency broadcast system. Works offline. SMS fallback. No internet required after initial sync.

### BuscaVida
Missing persons alert network compatible with existing Amber Alert systems. Face recognition optional. Privacy-first design.

### MapaSolidario
OpenStreetMap-based relief coordination. Mark needs, resources, volunteer positions. Sync across organizations.

## Self-Improving System
HumanAlert improves itself continuously:
- Every real alert processed trains the model
- Weekly automated test runs and quality checks
- Community contributions automatically reviewed and integrated
- Documentation auto-updated with every release

## Deployment
- Self-hosted: Docker Compose (recommended)
- Cloud: Works on any VPS ($5/month DigitalOcean sufficient for 10,000 users)
- Offline: Can run on a Raspberry Pi for local emergency networks
